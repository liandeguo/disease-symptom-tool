# Symptom Explorer
Search a database of 700+ diseases and instantly see which symptoms are associated with them.

## Why I built this
I attend lectures at the Medizinische Hochschule Hannover (MHH) and have a long-standing interest in how clinical data an be made more accessible through software. This tool started as a way to learn to work with medical datasets.

## Features
- Search across 700+ diseases by name
- View associated symptoms and their frequency per disease
- See recommended test procedures and treatment options

## Stack
Svelte · shadcn/ui · Vercel

## Data
Based on the [Disease and Symptoms dataset](https://www.kaggle.com/datasets/hagari/disease-and-their-symptoms/data) 
from Kaggle. The original dataset was cleaned and restructured for 
this application.
